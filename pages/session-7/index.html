üìñ Introduction to JavaScript Loops

What is a Loop?

A loop is a programming construct that allows you to repeat a block of code multiple times. Loops help you perform repetitive tasks efficiently and make your code more concise.

Types of Loops in JavaScript

  • For Loop: Repeats a block of code a specific number of times.
  • While Loop: Repeats a block of code as long as a condition is true.
  • Do-While Loop: Similar to the while loop, but guarantees the code block runs at least once.

üîç Exploring Loops

1. For Loop

The for loop repeats a block of code a specific number of times. It's useful when you know how many times you need to repeat the task.

for (let i = 0; i < 5; i++) {
  console.log("This is iteration number " + i);
}

2. While Loop

The while loop repeats a block of code as long as a specified condition is true. It's useful when you don't know how many times you need to repeat the task.

let i = 0;
while (i < 5) {
  console.log("This is iteration number " + i);
  i++;
}

3. Do-While Loop

The do-while loop is similar to the while loop, but it guarantees that the code block runs at least once, regardless of the condition.

let i = 0;
do {
  console.log("This is iteration number " + i);
  i++;
} while (i < 5);
